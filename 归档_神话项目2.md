## 一、项目概览

盘古是一个**主权可控的符号推理智能体**。不依赖任何外部 API，完全本地运行。v0.10.0 "超我" 对齐了6个社区标杆项目的核心能力，保持纯符号+零依赖的差异化优势。

### 核心能力

| 模块 | 能力 | 对齐 |
|------|------|------|
| **16种认知架构引擎** | CoT/ToT/MCTS/ReAct/苏格拉底/辩证等 | Agent Reasoning |
| **4D持久记忆** | 身份/陈述/技能/对话历史 | Hermes Agent |
| **梦境引擎** | 闲时自动反思+路径重放+规则归纳 | Shadow Brain |
| **知识图谱** | 实体提取+关系发现+混合搜索 | GBrain |
| **现实监督器** | 一致性验证+幻觉检测+可信度评分 | Evo Brain |
| **MCP协议桥接** | 7个工具方法/JSON标准输入输出 | NEXO |
| **自动技能创建** | 任务评估+自动创建/更新SKILL | Hermes Agent |
| **骨骼守护** | 多身份体系/3级响应 | 自有 |
| **符号推理引擎** | 合一+回溯+动态重排+因果链 | 自有 |
| **显式规则学习** | 自然语言命令动态添加规则 |
| **隐式规则学习** | 从正例归纳新规则 |
| **反例学习** | 交互式规则修正与特化 |
| **会话记忆** | FIFO 事实存储，参与推理 |
| **精准自我反思** | 失败时定位缺失事实 |
| **骨骼守护** | 主权硬编码，违词正则拦截 |
