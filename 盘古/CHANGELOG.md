# 变更日志

## [0.11.0] - 2026-07-02 "知行"
### 新增 - OpenClaw 配置体系对齐
- **ConfigLoader** — 零依赖 Markdown 配置文件解析器
  - `SOUL.md`：身份名称、版本、主权声明、价值观与原则
  - `AGENTS.md`：模块开关（dream_engine / knowledge_graph 等）及推理方法列表
  - `USER.md`：用户偏好（语言、输出格式、记忆上限、梦境间隔等）
  - 文件不存在时使用内置默认值，不抛出异常
  - `write_defaults()` 方法可自动生成模板文件
- **SuperBrainAgent** 在启动时自动加载配置，身份名称由 `SOUL.md` 驱动

### 新增 - ds4 本地推理引擎接口对齐
- **LocalInferenceEngine** — 可编程 Python API，与 MCP 协议桥接互补
  - `infer(goal, method)` — 单目标推理，支持 `method="auto"`（仲裁器自动选择）
  - `infer_all(goal)` — 16 种方法并行推理，返回结果对比
  - `learn(rule, force)` — 学习规则/事实（事实直接进 KB，无孤儿检查）
  - `remember(fact)` — 持久化事实到 4D 记忆并加入 KB
  - `recall(query)` — 从持久记忆召回相关条目
  - `search(query)` — 知识图谱混合搜索
  - `health()` — 知识库健康报告（含可信度评分）
  - `dream()` — 立即触发梦境反思
  - `soul_info()` / `agents_info()` / `user_prefs()` — 配置内省
  - 内置线程锁，线程安全
  - `stop()` 优雅关闭后台梦境引擎

### 新增 - 配置模板文件
- `SOUL.md` — 身份与价值观默认模板
- `AGENTS.md` — 模块配置默认模板
- `USER.md` — 用户偏好默认模板

### 测试
- 新增 `test_pangu_v0.11.0.py`（48 个测试，100% 通过）
  - ConfigLoader 全路径覆盖（默认值、文件加载、类型解析、write_defaults）
  - LocalInferenceEngine 全方法覆盖（推理、学习、记忆、搜索、健康、梦境）
  - SuperBrainAgent 配置集成测试
  - 版本标识验证

## [0.10.0] - 2026-06-13 "超我"
### 新增 - 16种认知架构
- **CoT** (Chain of Thought): 逐步推理链
- **ToT** (Tree of Thought): 多路径分支搜索
- **ReAct**: 推理与行动交替
- **MCTS** (Monte Carlo Tree Search): 随机模拟 + UCB
- **Socratic**: 苏格拉底式提问-回答-追问
- **Decomp**: 复杂目标拆解
- **Self-Refine**: 生成→评估→改进
- **Recursive**: 深度优先递归分解
- **Analogy**: 类比推理，找相似结构
- **Abductive**: 溯因推理，从结果推原因
- **Inductive**: 归纳推理，提取公共模式
- **Dialectic**: 正-反-合辩证推理
- **Counterfactual**: 反事实"如果...会怎样"
- **StepBack**: 步退抽象，整体视图
- **Contradiction**: 矛盾检测与自洽验证
- **Ensemble**: 集成投票，多数方法投票

### 新增 - 4D持久记忆 (Hermes对齐)
- 身份层 (IDENTITY.json)
- 陈述记忆 (MEMORY.json)
- 技能记忆 (SKILLS.json)
- 对话历史 (CONVERSATION.jsonl)
- 关键词匹配召回
- 1000条上限FIFO

### 新增 - 梦境引擎 (Shadow Brain对齐)
- 后台线程闲时自动反思
- 推理路径重放
- 规则一致性检查
- 自动归纳新规则
- 立即触发 `梦境` 命令

### 新增 - 知识图谱 (GBrain对齐)
- 纯Python dict + JSON存储
- 实体管理（类型+属性）
- 关系管理（主语-谓词-宾语-权重）
- 混合搜索（精确+语义）
- 导出为盘古事实

### 新增 - 现实监督器 (Evo Brain对齐)
- 事实一致性验证
- 逻辑一致性检查
- 否定矛盾检测
- 可信度评分
- 幻觉风险评估

### 新增 - MCP协议桥接 (NEXO对齐)
- 7个工具方法 (query/learn/reason/health/memory/search/dream)
- JSON标准输入输出协议
- 可被外部智能体调用

### 新增 - 自动技能创建 (Hermes对齐)
- 任务执行后评估
- 多次尝试成功自动创建技能
- 失败重试后成功也创建
- 技能去重与持久化

### 变更
- 骨骼守护升级为多身份体系（3级响应）
- 引擎整合为 SuperBrainAgent 主控
- 所有模块零外部依赖
- CLAUDE.md 流程优化：取消确认步骤，自动放行

## [0.9.0] - 2025-04-02 "自知"
### 新增
- 一致性证明器（孤儿规则、循环依赖、元数不一致、未定义谓词检测）
- 健康报告命令 `检查一致性`
- 推理轨迹与警告传播（强制添加的规则在答案后附加警告）
- 内置谓词 `eq/2`（至少一个常量）
- 规则添加时的强制选项（`force`）与不可靠标记

### 修复
- 隐式学习中的变量映射错误（链式变量共享）
- 动态事实重复合并问题
- 长链规则拒绝自动归纳（体长度≥8 转为用户确认或拒绝）
- 元数不一致拒绝加载（无 force 例外）

### 变更
- `KB.add_rule` 集成一致性检查，默认拒绝不一致规则
- 命令 `正确` 和 `错误` 分别触发隐式/反例学习
- 优化回溯引擎：动态重排基于子目标估算
