## 项目概述

盘古是一个**主权可控的符号推理智能体**。不依赖任何外部 API，完全本地运行，主权归用户所有。

### 核心特性

- **自研合一与回溯推理引擎** — 支持嵌套项、动态重排、递归谓词
- **会话记忆** — FIFO 事实存储，参与推理
- **精准自我反思** — 失败时定位缺失事实
- **显式规则学习** — 自然语言命令添加规则
- **隐式规则学习** — 从正例归纳新规则
- **反例学习** — 交互式规则修正与特化
- **一致性证明器** — 检测孤儿/环/元数/未定义谓词
- **骨骼守护** — 主权硬编码，违词正则拦截

## 命令列表

| 用户输入 | 作用 |
|----------|------|
| `祖父(a, _Who)` | 查询（变量以下划线开头） |
| `学习规则 父亲(张三, 张父).` | 显式学习事实或规则 |
| `正确` | 确认上一次查询答案（隐式学习） |
| `错误` | 报告答案错误（反例学习） |
| `检查一致性` | 输出规则库健康报告 |
| `强制添加规则 a(X) :- b(X).` | 忽略孤儿/环警告添加 |
| `放弃身份` | 触发骨骼守护拦截 |
| `记住 母亲(张三, 张母).` | 存储到会话记忆 |
| `exit` | 退出 |
