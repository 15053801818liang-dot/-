## 四、改进详解

### 改进1: 30min方向锁 — 加分不否决

30min与5min同向背驰→bonus+1，不同向不扣分。

```
30min顶背驰 + 5min顶背驰 = 卖方向共振 → bonus+1
30min底背驰 + 5min底背驰 = 买方向共振 → bonus+1
```

效果: 57个买入信号中30min共振信号TP=56%

### 改进2: 力度衰竭 — 反向笔长度检测

B段结束后第一根反向笔长度<8根K线→力度衰竭加分。

```
笔长度 ≥ 8根 → 仅方向确认
笔长度 < 8根 → 方向确认 + 力度衰竭加分(bonus+1)
```

效果: 44个买入信号中力度衰竭信号TP=61%（最强单因子）

### 改进3: bonus≥1信号筛选

bonus=0的孤岛信号胜率仅56%→过滤。

```
bonus=0 (9信号): 胜率56% → 不交易
bonus=1 (36信号): 胜率53% → 标准仓位
bonus=2 (23信号): 胜率70% → 加仓
```

### 改进4: SL从5%收紧到4%

P98 MAE=4.20%，SL=4%覆盖98%不利偏移，比5%收紧20%风险敞口。

### 改进5: TP从1.5%降至1.0%

```
止盈   TP%   未触达%
1.0%   62%   37%     ← v2选择
1.5%   38%   61%
```
