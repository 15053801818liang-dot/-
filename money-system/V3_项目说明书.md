## 八、与V1/V2的关键差异

| | V1/V2 共同特征 | V3 突破 |
|------|------|------|
| 盈亏比方向 | 窄TP宽SL → 保本线76%+ | 宽TP窄SL → 保本线22% |
| 胜率要求 | 需要65%+（实际达不到） | 仅需23%（实际54%，31%安全边际） |
| 数学本质 | 靠高胜率活（活不了） | 靠盈亏比活（活得很好） |
| 复杂度 | V2加闸门/结构止损 | 极简：纯阿娇信号 |
| 可证伪性 | V2闸门是过拟合 | V3只有一个可变参数(TP/SL比) |
